🏷️ YOLO 实时目标检测

YOLOv5 / YOLOv8 · 单阶段检测 · 速度 / 精度平衡 · 多任务扩展

为什么用 YOLO?
  • 实时性：高 FPS，适合边缘/摄像头场景。
  • 部署友好：轻松导出 ONNX / TensorRT / TFLite。
  • 社区生态：教程 & 预训练权重丰富。
  • 多任务：检测 + 分割 + 姿态 + 旋转框。
核心思想
  • 单阶段直接预测边界框和类别概率。
  • 跨尺度特征融合 (FPN / PAN)。
  • IoU / CIoU / DFL 损失提升定位质量。
与两阶段比较
  • 对比 Faster R-CNN：更快但对小目标略逊。
  • 对比 Anchor-Free：仍保持 Anchor 友好方案（v8 引入 task head 简化）。
  • v8 以后：改进类别/框解耦，提升泛化。
性能提示
  • 选择 s / m / l / x 平衡计算 & 精度。
  • 混合精度 (FP16) 可再提速 20~40%。
  • 导出 TensorRT 后延迟显著下降。
适用场景
  • 工业/安防实时检测
  • 无人机 / 交通监控
  • 轻量端到端推理 Demo
